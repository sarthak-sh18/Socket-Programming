The Project contains 2 main files and 1 folder i.e.
1. client.c (Compile using command "cc client.c -o client" and run using ./client)
2. server.c (Compile using command "cc server.c -o server" and run using ./server)

The folder named db contains all the data whcih is related to Login(Containing Login Data of Agent, Customer and Admin), Train Information and booking information.
